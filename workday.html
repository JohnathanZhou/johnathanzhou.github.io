Software Engineering Intern, Summer 2018

This past summer, I worked on a user profile integration for Workday's Automated Regression Platform (WARP). I worked on ideation through release of the user profile integration, conducting user interviews, creating lo-fidelity mockups, and more.

My work consisted of learning Scala & the WARP codebase, designing an updated database schema, then completing the backend/frontend functionality necessary for user profile interactions. Some technologies I worked with include MySQL, the Play Framework, the pac4j security library, Twirl templating, Slick, and Bootstrap.

Ultimately, my work will cut down test lookup time immensely and also save the average WARP user at least 3 clickthroughs per session.
